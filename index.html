Types of web development

Web development can be broken down into three layers: client-side coding (frontend), server-side coding (backend) and database technology.


Client-side (Frontend)

Client-side scripting, or frontend development, refers to everything that the end user experiences directly. Frontend code executes in a web browser and directly relates to what people see when they visit a website. Things like layout, fonts, colours, menus and contact forms are all driven by the frontend.
