Frontend Development:

   Frontend development focus on the structure and the design of the website.A front-end developer uses code that affects how a website looks and how a user interacts with its webpages.
That includes simple elements such as buttons that take a visitor to different places within the website, as well as input methods, like a contact form.
The programming languages that a front-end developer uses can create a static webpage or a dynamic one. Programming languages they use include:
Languages used for Frontend Development are:
  • HTML
  • CSS
  • JavaScript
